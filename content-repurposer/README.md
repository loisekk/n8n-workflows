# AI Content Repurposer

Transform content across multiple platforms using AI.

## Setup

1. Import `workflows/ai-content-repurposer.json` into n8n
2. Configure OpenAI or Ollama credentials
3. Activate the workflow

## How It Works

1. Receives input content
2. AI processes and reformats for target platform
3. Outputs platform-optimized content

## Required Credentials

- OpenAI API or Ollama (local)
