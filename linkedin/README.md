# LinkedIn Automation

Automate LinkedIn post creation and scheduling with AI.

## Setup

1. Import `workflows/automate-linkedin-posts-with-ai.json` into n8n
2. Configure AI and LinkedIn credentials
3. Activate the workflow

## How It Works

1. Receives topic or content idea
2. AI generates LinkedIn-optimized post
3. Formats for maximum engagement
4. Posts or schedules to LinkedIn

## Required Credentials

- OpenAI API or Ollama (local)
- LinkedIn API
