# Automated Meeting Notes

Generate and distribute meeting notes automatically.

## Setup

1. Import `workflows/automated-meeting-notes.json` into n8n
2. Configure required credentials
3. Activate the workflow

## How It Works

1. Receives meeting transcript or recording
2. AI extracts key points and action items
3. Formats into structured notes
4. Distributes to participants

- OpenAI API or Ollama (local)
