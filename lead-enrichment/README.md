# Lead Enrichment Pipeline

Automatically enrich and qualify leads with AI-powered data extraction.

## Setup

1. Import `workflows/lead-enrichment-pipeline.json` into n8n
2. Configure required API credentials
3. Activate the workflow

## How It Works

1. Receives lead data
2. Enriches with additional information
3. Qualifies and scores leads
4. Routes to appropriate channel
